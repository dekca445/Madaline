Slide 2 — Fondasi Teori

ADALINE & MADALINE

  • ADALINE (Adaptive Linear Neuron) — unit jaringan saraf dasar oleh Widrow & Hoff (1960). Menggunakan net input langsung untuk pembelajaran.
  • MADALINE (Many ADALINE) — menggabungkan beberapa unit ADALINE dalam hidden layer untuk menyelesaikan masalah klasifikasi non-linear.
  • Arsitektur kami: 3 Input → 2 ADALINE → 1 Output dengan AND Gate untuk deteksi penyakit flu.
  • Fungsi Aktivasi Bipolar: Menghasilkan output +1 (Flu) jika net input ≥ 0, atau -1 (Tidak Flu) jika di bawah 0.

Slide 3 — Arsitektur Jaringan

Arsitektur MADALINE untuk Deteksi Flu

  • Input Layer: 3 neuron untuk fitur gejala — Demam (X₁), Batuk (X₂), Pilek (X₃). Nilai bipolar: +1 = Ya, -1 = Tidak.
  • Hidden Layer: 2 unit ADALINE. Masing-masing memiliki 3 bobot (w₁, w₂, w₃) dan 1 bias (b).
  • Output Layer: 1 neuron dengan AND Gate. Y_out = +1 jika kedua ADALINE = +1.
  • AND Gate: Jika salah satu ADALINE bernilai -1, maka output = -1. Output = +1 hanya jika kedua ADALINE = +1.
