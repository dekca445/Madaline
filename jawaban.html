Cloud Sync — GitHub Gist

Agar pertanyaan dari HP muncul di laptop, konfigurasi cloud sync menggunakan GitHub Gist. Data Q&A akan disimpan di Gist dan bisa diakses dari perangkat manapun.

Cara Setup: 1. Buka gist.github.com → buat Gist baru dengan nama file madaline_qa.json dan isi []
2. Copy Gist ID dari URL (bagian akhir setelah username/)
3. Buat token di GitHub Settings → Fine-grained tokens dengan permission "Gists: Read and Write"
4. Masukkan keduanya di bawah ini, lalu klik Simpan
